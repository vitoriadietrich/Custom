# Custom Radio Button in HTML and CSS

Um projeto simples de **radio button personalizado** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar **radio buttons customizados**, substituindo o estilo padrão por um design moderno e estilizado.

## 🛠️ Tecnologias

- HTML
- CSS

## 🎯 Funcionalidades

- Radio button estilizado
- Design moderno
- Fácil de integrar em formulários

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
